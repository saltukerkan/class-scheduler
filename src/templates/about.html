About this project

When the time came to choose classes among many alternative electives and even more sections, I always struggled to build the perfect schedule. So I decided to write a program to show me a list of possible schedules given the constraints (mandatory sections and such). Then I decided to take it to the next by including a user interface, and I wanted to get into web programming, so this project was born.
With this application you can select all the classes you have to take and add some electives, and you will be shown all the possible schedules. All you will have to do is select your favourite schedule and you will be given the CRN's you need to enter to your school's registeration system.
